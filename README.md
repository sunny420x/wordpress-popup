# Wordpress Popup System #

ระบบสร้างและจัดการ Popup เช่น โมษณา สิทธิพิเศษ Artwork เทศกาลต่าง ๆ เป็นต้น

## ระบบ Popup นี้ สามารถทำอะไรได้บ้าง ? ##

สามารถสร้าง Profile Popup มากกว่าหนึ่ง profile และ เลือกเปิด/ปิด ตามโอกาสต่าง ๆ ได้ โดยสามารถเลือกโชว์ได้เพียงแค่ 1 popup เท่านั้น

## วิธีติดตั้งปลั้กอิน ##
ไปที่ <code>/wp-admin/plugin-install.php</code> และอัปโหลดไฟล์ zip ของปลั้กอิน จากนั้นสามารถ <b>Activate</b> แล้วเริ่มใช้งานปลั้กอินจากเมนูได้เลย !
